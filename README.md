# Portfolio Optimization via Stochastic Portfolio Theory (SPT)

A research-level implementation of multiple portfolio optimization strategies on Indian equity data (Nifty 50 universe), progressing from classical baselines to neural network-driven SPT models inspired by **Kom Samo & Vervuurt (2016)**.

---

## Methods Implemented

| Method | Type | Loss Function |
|---|---|---|
| Equal Weight | Baseline | No optimization |
| Static Markowitz | Classical | Max Sharpe (analytical) |
| Rolling Markowitz | Classical | Max Sharpe (rolling window) |
| ML Portfolio (SPTModel) | Neural Network | Sharpe + Drawdown + L2 |
| Classical SPT (Diversity Weighted) | Analytical | No optimization |
| Parametric SPT | Learned scalar p | Relative Sharpe Loss |
| Functional SPT | Deep Network + Residual | Relative Log-Wealth + Sharpe + Downside + Concentration |

---

## Loss Functions

### 1. Equal Weight Portfolio
```
No loss function.
Fixed weights: w_i = 1/N for all stocks
Serves as a naive diversification baseline.
```

---

### 2. Static Markowitz (Max Sharpe)
```
Objective: Maximize Sharpe Ratio

         μᵀw - r_f
max   ────────────────
 w     √(wᵀΣw)

where:
  μ  = EMA historical expected returns
  Σ  = Sample covariance matrix
  w  = Portfolio weights
  r_f = Risk-free rate (default 0)

Constraint: Σw_i = 1,  0 ≤ w_i ≤ 0.4
Solved analytically via PyPortfolioOpt (EfficientFrontier)
```

---

### 3. Rolling Markowitz (Out-of-Sample)
```
Same objective as Static Markowitz but solved on a
rolling 252-day training window, rebalanced every 21 days.

Objective (per window):

         μ_window ᵀw - r_f
max   ──────────────────────
 w      √(wᵀ Σ_window w)

where μ_window and Σ_window are estimated
from the most recent 252 trading days only.
```

---

### 4. ML Portfolio (SPTModel) — Neural Network
```
Loss = -Sharpe(Rp) + λ₁ · L2_penalty + λ₂ · MaxDrawdown_penalty

where:

  Rp = Σ w_i · r_i          (portfolio return per day)

  Sharpe(Rp) = E[Rp] / (σ[Rp] + ε)

  L2_penalty = Σ ||θ||²      (weight regularization)
               θ ∈ model

  MaxDrawdown_penalty:
    cumulative  = Π(1 + Rp)
    rolling_max = cummax(cumulative)
    drawdown    = (rolling_max - cumulative) / rolling_max
    MaxDD       = max(drawdown)

Hyperparameters:
  λ₁ (l2_lambda)  = 1e-4
  λ₂ (dd_lambda)  = 0.1

Model: 3-layer MLP with BatchNorm + Dropout
       Input → 64 → 32 → 1 → Softmax(weights)
```

---

### 5. Classical SPT — Diversity Weighted Portfolio
```
No loss function — closed-form analytical solution.

Weights are generated by:

  w_i = μ_i^p / Σ μ_j^p

where:
  μ_i = market weight of stock i (price-based proxy)
  p   = diversity parameter ∈ (0, 1)

Tested with p = 0.5 and p = 0.8
Lower p → more diversification away from large caps
```

---

### 6. Parametric SPT — Learned p
```
Loss = -RelativeSharpe(Rp, Rm)

where:

  Rp = Σ w_i(p) · r_i       (portfolio return)
  Rm = Σ μ_i · r_i           (market return)

  RelativeSharpe = E[Rp - Rm] / (σ[Rp - Rm] + ε)

  w_i(p) = μ_i^p / Σ μ_j^p  (diversity weights)

  p is a LEARNABLE scalar parameter initialized at 0.5
  Model learns the optimal diversity exponent p
  by maximizing relative Sharpe over market.

Optimizer: Adam (lr=0.05), 300 epochs
```

---

### 7. Functional SPT — Deep Network (Kom Samo & Vervuurt Inspired)
```
Loss = -Objective + λ_smooth · SmoothnessP enalty

Objective = RelLogWealth
          + λ_sharpe · Sharpe(Rp)
          - γ · DownsidePenalty
          - 0.1 · ConcentrationPenalty

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Term 1 — Relative Log-Wealth (SPT Core):
  Rp = Σ w_i · r_i
  Rm = Σ μ_i · r_i
  RelLogWealth = E[log(1 + Rp) - log(1 + Rm)]

  → Maximizes compounded growth over market portfolio

Term 2 — Sharpe Component:
  Sharpe = E[Rp] / (σ[Rp] + ε)

  → Rewards risk-adjusted returns, not just raw returns

Term 3 — Downside Penalty:
  DownsidePenalty = E[clamp(Rp, max=0)²]

  → Penalizes negative returns quadratically
  → Losses are punished harder than equivalent gains

Term 4 — Concentration Penalty (Herfindahl Index):
  ConcentrationPenalty = E[Σ w_i²]

  → Penalizes weight concentration in single stocks
  → Forces diversification across the universe

Term 5 — Smoothness Penalty:
  x_sorted = sorted rank values
  f_vals   = model.f(x_sorted)
  Δ¹       = f_vals[1:] - f_vals[:-1]   (first difference)
  Δ²       = Δ¹[1:] - Δ¹[:-1]          (second difference / curvature)
  SmoothnessP enalty = E[Δ²²]

  → Ensures weight function f(rank → weight) is smooth
  → Prevents erratic weight jumps between similar-ranked stocks
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Hyperparameters:
  λ_sharpe    = 0.5
  γ           = 3.0   (downside weight)
  λ_smooth    = 3.0   (smoothness weight)
  0.1                 (concentration weight)

Model Architecture:
  Input: rank score per stock (scalar)

  Main branch:    Linear(1→32) → Tanh → Linear(32→32) → Tanh
  Residual:       Linear(1→32)
  Output branch:  Linear(32→16) → Tanh → Linear(16→1) → Softplus
  Normalization:  weights = f_vals / sum(f_vals)

Training:
  Optimizer : Adam (lr=0.001)
  Scheduler : CosineAnnealingWarmRestarts (T_0=200, T_mult=2)
  Grad Clip : max_norm=1.0
  Early Stop: patience=200 on objective
  Max Epochs: 2000
```
